* Uses the ZH-L16 Algorithm
* **Skills:** C#, Unity, Algorithms, Physics

The equation for ZH-L16 is as follows:
<br>
**P = Pio + R(t - 1/k) - (Pio - Po - (R/k)) e<sup>-kt</sup>**
